# Crime data EDA project README

This project is a part of Rockborne’s python EDA project and involves analysing multiple police forces across different regions in UK in python to present a report of the results to the head of sales for a real estate company

This project was planned and managed using a trello board :
https://trello.com/b/xfd6Rkxc/python-crime-data-eda

All changes were tracked using Git and recorded in a GitHub repository:
https://github.com/FreyaClifford/Crime_data_EDA

#Needed Libraries 
These python libraries were used to complete the project and should be imported before running any code
- pandas
- numpy
- geopandas
- folium

#Files included 
- Crime_EDA_work_statement - Statement of work
- Data folder - includes the raw and cleaned csv files as well as UK district boundaries used for visualising maps
- Crime_EDA_Cleaning - Jupyter notebook of data cleaning and preprocessing (run before analysis notebook)
- Crime_EDA_Analysis - Jupyter notebook of data eda and visualisations (run after cleaning notebook)
- Crime_EDA_Cleaning_Flowchart - screenshot of the workflow chart visualising the data cleaning process
- Crime_EDA_Report -  Final report of outcome of project to be presented to stakeholders
